TOOL-001 · PointTool
字段	内容
工具类型	标注工具
适用任务类型	关键点标注
适用模态	图像
功能描述	用于人体/手部/物体关键点标注；支持自定义关键点列表和顺序；支持图片缩放和点拖拽调整
负责团队	内部工具团队
访问地址	https://标注平台/point-tool
文档链接	内部Wiki-PointTool使用指南
适用规则ID	RULE-009、RULE-010
配置状态	已配置
当前版本	V3.2
备注	支持21点手部和24点人体关键点标注
输出格式说明
每张图输出一个JSON对象；width/height记录图片尺寸；step_1.result数组记录每个关键点；每个点含x/y坐标、attribute标签、valid是否有效、order序号、label关键点名称。
输出格式JSON示例
json
{
  "width": 1920,
  "height": 1080,
  "rotate": 0,
  "valid": true,
  "step_1": {
    "dataSourceStep": 0,
    "toolName": "pointTool",
    "result": [
      {
        "x": 670.07,
        "y": 617.71,
        "attribute": "",
        "valid": true,
        "id": "LV0iF3Ve",
        "sourceID": "",
        "textAttribute": "",
        "order": 1,
        "label": "0_pelvis"
      },
      {
        "x": 770.70,
        "y": 605.76,
        "attribute": "",
        "valid": true,
        "id": "vHnkZugJ",
        "sourceID": "",
        "textAttribute": "",
        "order": 2,
        "label": "1_left_hip"
      }
    ]
  }
}
